# ♻️ ReWear – Community Clothing Exchange Platform

ReWear is a full-stack MERN application that promotes sustainable fashion by enabling users to exchange, donate, and redeem pre-owned clothing. The platform helps reduce textile waste through a community-driven clothing reuse ecosystem.

## 🚀 Features

* Secure JWT Authentication
* User Profile Management
* Clothing Item Listings with Image Uploads
* Browse and Search Available Items
* Swap Request System
* Point-Based Item Redemption
* Admin Moderation & Approval
* Responsive and User-Friendly Interface

## 🛠️ Tech Stack

**Frontend:** React.js, Tailwind CSS, Axios

**Backend:** Node.js, Express.js

**Database:** MongoDB, Mongoose

**Authentication:** JWT

**File Uploads:** Multer

## 📂 Project Structure

```bash
frontend/   # React Frontend
backend/    # Express Backend
```

## ⚙️ Installation

```bash
git clone <repository-url>

# Backend
cd backend
npm install
npm start

# Frontend
cd frontend
npm install
npm run dev
```

## 🌱 Objective

ReWear encourages sustainable fashion by providing a platform where users can give a second life to unused clothing through swapping and point-based exchanges.
